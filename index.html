Detailed help for LEGO® BOOST app programming blocks extracted directly from the app

Follow these steps:

  1. Download the LEGO® BOOST app's OBB from your favorite APK mirror.

    Example URLs:
    • APKCombo
    • APKPure

    The current version (1.9.2) is 439.2 MB.

  2. If you downloaded a packed version (.zip or .xapk), then get the OBB: 
          unzip -j <downloaded_file_name> '*main.*.com.lego.boost.boost.obb'
  3. Extract help messages for the supported languages from the OBB: 
          unzip -j main.*.com.lego.boost.boost.obb 'assets/BlockHelp/*.json'

    The files will be named according to the locale naming conventions with an extension of .json.

  4. Choose from the extracted language files in the file selector:

    The images will be loaded from lego.com, where some of them are not available, some images are not even defined. Some of the blocks are not available in the app, and some of them are available but not included in these lists.

  5. You can hide these steps. Reload the page to redisplay.
